Optimizer

  • As we mentioned in the previous step of the guide, the learning process of a neural network consists of repeatedly adjusting the network parameters (weights and biases) in order to minimize the difference between the predicted label (output) of the network and the actual label from the training dataset. That difference represents the error of the neural network, and it is computed with the help of a loss function. In other words, the adjusting of the network parameters is done with the intent to minimize the cost function. Therefore, a loss function's role is to evaluate the network's error.

  • Mean Squared Error

    Mean Squared Error (or MSE) is computed by averaging the squared differences between the predicted labels and the actual labels. If we had \(n\) training instances and labels \(t\), for which the network predicted \(n\) outputs \(y\), then the formula for the MSE is: \[ MSE = \frac{1}{2n} \sum_{i}^{n}(t_i - y_i)^{2} \]

    One thing to note about the MSE is that its formula resembles the one for the variance, which computes how far some elemets are from the mean.

    The Mean Squared Error is continuous and easily differentiable, which will be very useful in the training process, since the Backpropagation algorithm uses the gradient of the loss function extensively in the training process: \[ MSE' = -\frac{1}{n} \sum_{i}^{n}(t_i - y_i)y_i' \] where \(y_i'\) is the gradient of the activation function.
